How this works

  • 14 features: home/away points/game scored and allowed, yards per play, turnover rate, QB rating, PPG differential, defense differential, home-field advantage, rest days.
  • 3 XGBoost models: classifier (win probability), regressor (spread), regressor (total).
  • Default model: trained on 400 synthetic games so the dashboard works offline. Replace with a production-trained model via model_path (output of NFLGamePredictor.save()).
  • CLI equivalent: sportsquant nfl predict-game --home KC --away BAL --season 2024 --week 10
